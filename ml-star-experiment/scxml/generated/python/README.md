# pipchannelstatemachine

SCXML state machine generated by [SCXML-Gen](https://github.com/miho/scxml-gen).

## Installation

```bash
pip install -e .
```

Or install dependencies directly:

```bash
pip install dukpy>=0.3.0
```

## Usage

```python
from pip_channel_state_machine import PipChannelStateMachine

# Create and start the state machine
machine = PipChannelStateMachine()
machine.start()

# Send events
machine.send("my_event")

# Check state
if machine.is_in_state("active"):
    print("Machine is active")

# Check if finished
if machine.is_finished:
    print("Machine has finished")
```

## Using the Executor (Thread-Safe, Autonomous Timers)

`ContinuousExecutor` runs on a background thread and automatically processes
delayed events (`<send delay="..."/>`) — no manual `pump_events()` calls
required. Timers fire at the exact scheduled time via `threading.Timer`.

```python
from scxmlgen.executor import ContinuousExecutor
from pip_channel_state_machine import PipChannelStateMachine

machine = PipChannelStateMachine()

with ContinuousExecutor(machine) as executor:
    executor.start()

    # Send events (thread-safe)
    executor.send("event1")

    # Awaitable sending
    future = executor.send_async("event2")
    future.result()  # wait for processing

# Executor is shut down automatically by the context manager
```

## Tracing

```python
from scxmlgen.trace import JsonlTraceWriter
from pip_channel_state_machine import PipChannelStateMachine

# Create trace writer
with JsonlTraceWriter("trace.jsonl") as writer:
    machine = PipChannelStateMachine()
    machine.set_trace_listener(writer)
    machine.start()
    machine.send("go")
```
